## How to Convert to PDF

### Option 1: Using Markdown to PDF Converter

**Online Tools:**
1. Visit: https://www.markdowntopdf.com/ or https://md2pdf.netlify.app/
2. Copy the content from the `.md` file
3. Paste and convert to PDF
4. Download and print

**Command Line (if you have tools installed):**
```bash
# Install markdown-pdf (if not already installed)
npm install -g markdown-pdf

# Convert to PDF
markdown-pdf docs/WORKER_BREAK_ROOM_POSTER.md -o docs/WORKER_BREAK_ROOM_POSTER.pdf
markdown-pdf docs/SUPERVISOR_BREAK_ROOM_POSTER.md -o docs/SUPERVISOR_BREAK_ROOM_POSTER.pdf
```

### Option 2: Using Word/Google Docs

1. Open the `.md` file in a text editor
2. Copy all content
3. Paste into Microsoft Word or Google Docs
4. Format as needed:
   - **Font:** Arial or Helvetica, 11-12pt
   - **Heading 1:** 18-20pt, Bold
   - **Heading 2:** 14-16pt, Bold
   - **Body:** 11-12pt
5. Save/Export as PDF

### Option 3: Using VS Code with Extension

1. Install "Markdown PDF" extension in VS Code
2. Open the `.md` file
3. Right-click â†’ "Markdown PDF: Export (pdf)"
4. PDF will be generated in the same folder

- **Worker Poster:** Post in all factory break rooms
- **Supervisor Poster:** Post in supervisor offices, management areas, and break rooms
